### 1.2 Data Points Collected

| Sensor/System | Data Points | Frequency |
|---------------|-------------|-----------|
| Engine ECU | RPM, coolant temp, throttle position, fuel level | Every 5 sec |
| Battery BMS | Voltage, current, state of charge, temperature | Every 10 sec |
| TPMS | Tire pressure (all 4), temperature | Every 30 sec |
| GPS Module | Latitude, longitude, speed, heading | Every 5 sec |
| Accelerometer | X, Y, Z acceleration (driving behavior) | Every 1 sec |
| OBD-II DTCs | Diagnostic Trouble Codes (errors) | On change |

### 3.2 Agent Responsibilities

| Agent | Input | Output |
|-------|-------|--------|
| **Diagnosis Agent** | Telemetry + DTC codes | Component health assessment |
| **Scheduling Agent** | Predictions + User calendar | Optimal service appointment |
| **RCA/CAPA Agent** | Failure history + Patterns | Root cause + Corrective actions |
| **Feedback Agent** | User feedback + Service results | Quality improvement suggestions |
| **Customer Engagement** | All above + User preferences | Personalized notifications |

## 6. Demo vs Production

| Aspect | Demo (Current) | Production |
|--------|----------------|------------|
| Data Source | JSON files + Seed data | OBD-II devices via MQTT |
| Frequency | Manual/Simulated | Real-time (1-30 sec intervals) |
| ML Model | Pre-trained demo | Continuously learning |
| Alerts | Simulated | Real-time notifications |
| Scale | Single user | Millions of vehicles |

---

## 7. Hardware Requirements (Production)

For real deployment, vehicle owners need:

1. **OBD-II Telematics Device** (~$50-200)
   - Examples: Bouncie, Vyncs, Automatic
   - Plugs into vehicle's OBD-II port
   - Has cellular (4G) or WiFi connectivity

2. **Mobile App**
   - Pairs with OBD-II device
   - Receives push notifications
   - Shows real-time telemetry

3. **Optional: Advanced Sensors**
   - Dashcam with AI (accident detection)
   - Additional tire sensors
   - Fuel quality sensor

---

## 8. Security Considerations

- **Vehicle Authentication**: Each OBD-II device has unique certificates
- **Data Encryption**: TLS 1.3 for all transmissions
- **UEBA**: Detects anomalous patterns (theft, unauthorized access)
- **Data Privacy**: User controls what data is shared
