## Design System Summary

### Colors
- **Primary**: #3B82F6 (Blue)
- **Success**: #10B981 (Green)
- **Warning**: #F59E0B (Amber)
- **Danger**: #EF4444 (Red)
- **Background**: #0F172A (Dark)
- **Surface**: #1E293B (Card)
- **Text Primary**: #F8FAFC
- **Text Secondary**: #94A3B8

### Typography
- **Headings**: Inter Bold
- **Body**: Inter Regular
- **Monospace**: JetBrains Mono (for VINs, IDs)

### Spacing
- Base unit: 4px
- Padding: 16px (cards), 24px (page)
- Gap: 16px (grid), 8px (inline)

### Components
- Cards with 8px border radius
- Buttons with 6px border radius
- Inputs with 4px border radius
- Shadows for elevation
